## Run Locally

Clone the project

```bash
  git clone https://github.com/Trevor-Kayeyia-Madara/trevor-madara-kayeyia-portfolio.git
```

Go to the project directory

```bash
  cd trevor-madara-kayeyia-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
